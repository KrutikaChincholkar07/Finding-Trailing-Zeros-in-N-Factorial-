# Finding-Trailing-Zeros-in-N-Factorial-
This program finds the number of trailing zeros (ending zeros) in the factorial of a number (N!) entered by the user. Trailing zeros appear in a factorial result because of factors of 10, which are formed by multiplying 2 × 5. Since factorials contain more factors of 2 than 5, we only count the number of 5s in the factorial.
